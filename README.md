### 软件架构

后端：thinkphp8 管理端页面：element-plus 小程序端：uniapp。

部署环境建议：Linux + Nginx + PHP8.2 + MySQL5.7，上手建议直接用宝塔集成环境。

### 技术特点
- 前后分离 (分工协助 开发效率高)
- 统一权限 (前后端一致的权限管理)
- uniapp (一套代码8个平台，开发不浪费)
- thinkphp8(上手简单，极易开发)
- element-ui(饿了么前端开源管理后台框架，方便快速开发)
